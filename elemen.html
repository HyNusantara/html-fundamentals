Elemen HTML - HyNusantara Academy

Yuk kita kenalan dengan elemen-elemen yang ada di HTML. Berikut adalah contoh elemen HTML.

Text Elements

Text Element atau Elemen Teks pada HTML adalah yang paling penting. Karena paling banyak digunakan pada pembuatan web. Ada beberapa elemen teks yang bisa kita gunakan, antara lain:

  1. Heading
  2. Paragraph
  3. Formatting
  4. Comments
  5. Quotations
  6. Lists
  7. Links
  8. Web

Object Elements

Object Element pada HTML adalah elemen yang menghasilkan visual selain teks biasa. Berikut beberapa elemen yang bisa kita gunakan:

  1. Images
  2. Tables
  3. Block and Layouts
  4. Forms
  5. Media
  6. Graphic (SVG dan Canvas)
  7. IFrame

Attribute

Selain mengenal tentang elemen-elemen HTML, kita juga harus tahu tentang attribute pada elemen-elemen HTML. Attribute dapat diterapkan hampir disemua elemen, tapi ada beberapa attribute yang hanya berlaku untuk elemen tertentu saja, misalnya:

  1. Images
  2. Tables
  3. Forms
  4. Media
  5. Lists
  6. Links

Ini adalah contoh penggunaan attribute pada link.
